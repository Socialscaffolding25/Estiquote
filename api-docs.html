Estiquote API

Integrate UK home improvement cost data directly into your own tools, CRMs, calculators, or client-facing applications. The Estiquote API is available on the Trade plan.

Overview

The Estiquote API provides programmatic access to our UK construction cost database. It returns the same data that powers the Estiquote estimator tool — regional labour rates, material cost indices, and trade-specific pricing — in a machine-readable JSON format.

Base URL:

https://api.estiquote.co.uk/v1

All responses are returned as JSON. All requests must be made over HTTPS. HTTP requests will be refused.

📋 JSON responses 🔒 HTTPS only 🌍 REST API ⚡ ~120ms avg latency

Authentication

All API requests must include your API key in the Authorization header using Bearer token format.

Your API key:

Dashboard →
# All API requests use Bearer authentication curl https://api.estiquote.co.uk/v1/estimate \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json"

⚠ Keep your API key private. Do not expose it in client-side JavaScript, mobile apps, or public repositories. If your key is compromised, regenerate it from your dashboard.

GET /estimate

Returns an indicative cost estimate for a home improvement project based on trade type, region, area, and specification level.

GET /v1/estimate Returns cost range + breakdown

Query parameters

ParameterTypeDescription
traderequiredstringTrade category. One of: extension, loft, kitchen, bathroom, electrical, plumbing, structural, decoration
regionrequiredstringUK region. One of: london, south_east, south_west, midlands, north_england, scotland_wales
arearequirednumberProject area in m². Integer between 1–500.
qualityoptionalstringSpecification level: budget, mid, premium. Default: mid
extrasoptionalstringComma-separated extras: planning, structural, warranty, pwall
postcodeoptionalstringUK postcode for postcode-level pricing (overrides region if provided)

Example request

curl -G https://api.estiquote.co.uk/v1/estimate \ -H "Authorization: Bearer eq_live_..." \ -d "trade=extension" \ -d "region=south_east" \ -d "area=30" \ -d "quality=mid" \ -d "extras=planning"

Example response

{ "success": true, "estimate": { "range_low": 55400, "range_high": 80600, "currency": "GBP", "vat_included": false, "breakdown": { "labour": 36200, "materials": 26300, "prelims": 8100, "extras": 3500 }, "per_sqm": 2358, "contingency_15pct": 10500 }, "inputs": { "trade": "extension", "region": "south_east", "region_multiplier": 0.90, "area_sqm": 30, "quality": "mid", "extras": ["planning"] }, "meta": { "data_updated": "2025-01-01", "sources": ["BCIS", "BMF", "ONS"], "request_id": "req_7xK9mP2nQ" } }

GET /materials

Returns the current UK construction material price index with quarterly and year-on-year changes.

GET /v1/materials Returns material price index
curl https://api.estiquote.co.uk/v1/materials \ -H "Authorization: Bearer eq_live_..."

Example response

{ "success": true, "updated": "2025-01-01", "materials": [ { "id": "structural_timber_c16", "name": "Structural timber C16", "unit": "per m³", "price_range": { "low": 280, "high": 340 }, "change_qoq_pct": 5.1, "change_yoy_pct": 9.3 } // ... 9 more materials ] }

GET /regions

Returns all available UK regions with their labour cost multipliers relative to London.

GET /v1/regions Returns regional multipliers
{ "success": true, "regions": [ { "id": "london", "name": "London", "multiplier": 1.00 }, { "id": "south_east", "name": "South East England", "multiplier": 0.90 }, { "id": "south_west", "name": "South West England", "multiplier": 0.82 }, { "id": "midlands", "name": "Midlands", "multiplier": 0.78 }, { "id": "north_england", "name": "North of England", "multiplier": 0.74 }, { "id": "scotland_wales","name": "Scotland & Wales", "multiplier": 0.72 } ] }

Error Codes

HTTP statusError codeMeaning

401unauthorizedMissing or invalid API key
403plan_requiredAPI access requires Trade plan
